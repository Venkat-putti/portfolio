Personal Portfolio Website
HTML CSS JavaScript
Overview

I developed a personal portfolio website to present my skills, projects, and background in a structured and professional way. The goal was to create a clean, responsive, and user-friendly interface.

My Approach

I started by planning the layout and identifying key sections such as Home, About, Skills, and Projects. I focused on maintaining consistency across pages by designing a reusable navigation bar and a minimal UI.

During development, I ensured responsive design using flexible layouts and tested the website across different screen sizes. I also refined the color scheme to maintain a professional appearance.

Key Learning

This project helped me improve my front-end development skills, UI design thinking, and the importance of clean and maintainable code structure.

Music Web Application
Python Django JavaScript
Overview

I built a music web application that allows users to explore, play, and manage music content. The application is designed to provide a smooth and engaging user experience.

My Approach

I began by designing the backend using Django to handle user data, music storage, and application logic. I structured the project using the MVC pattern to keep the code organized and scalable.

On the frontend, I worked on creating an intuitive interface for browsing and playing music. I integrated dynamic features using JavaScript to enhance user interaction.

Key Learning

Through this project, I gained experience in full-stack development, backend architecture, and integrating frontend with server-side logic.
